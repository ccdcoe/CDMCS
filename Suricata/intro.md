# Course intro

* Designed as hands-on classroom course
* *from techies for techies*

* No slides, only markdown, Vagrantfiles and instructors in classroom
* Taking it remote

## Layout

* 6x3h sessions
* spread out over 3 weeks
* to avoid burnout
* shortened format
